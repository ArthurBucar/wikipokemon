# wikipokemon
Small project using API from https://pokeapi.co/, retrofit and anothers features.

EN - 
This is a small project in Kotlin that receives an API (https://pokeapi.co/) listing the pokemon and their skills.
I used retrofit to make my life easier and I did a small unit test using JUnit 4 and UI to mobile orientation changes.
Very basic project with a SplashScreen and Two Activities:
1 - First List the Pokemons
2 - List the photo and skill of the Pokemon

PT -

usei retrofit para facilitar minha vida e fiz um pequeno teste unitário usando JUnit 4 e IU se adapta as orientações do celular.
